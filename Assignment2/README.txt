Instructions for Assignment 2 Code

1) The analysis is performed using python3.7 and the requires libraries are:
	numpy(1.17.2 version), pandas(0.25.1 version), matplotlib(3.1.1 version) and mlrose_hiive(2.1.3   version)

2) All the code, graphs, datasets used and report are accessible through:
	https://github.com/ClaireShi28/MachineLearning/tree/master/Assignment2
   To run the code, please download the folder "Code" and run different python files for corresponding optimization problems.

3) The data used for this study are from UCI Machine Learning Repository and can be accessed from Data folder.
   There is no need to download data to run the code as data are loaded from url.
    
4) The code is modified based on mlrose manual:
  https://mlrose.readthedocs.io/en/stable/index.html
